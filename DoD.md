# Definition of Done (DoD) - BlueWallet

- Código revisado por pelo menos 1 membro do time
- Funcionalidade testada manualmente
- Critérios de aceitação da user story atendidos
- Sem erros aparentes na tela
- Commit feito com mensagem padronizada (feat:, fix:, docs:)
- Documentação atualizada (README.md ou docs/)
